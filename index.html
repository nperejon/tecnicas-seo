Importância do Title

  • Principal Head Tag
  • Ideal estar em 40 a 65 caracteres
  • Palavras-Chave
  • Diversidade de Title

Description Tag

  • Resumo
  • Máximo de 155 caracteres
  • Não é relevante para o motor de busca
  • É relavante ao usuário
  • Aparece abaixo do title e link no Google

Robots.txt e Meta Robot

  • Conversa com o mecanismo de busca
  • Presença essencial no seu site
  • Controle de Acesso
  • Controle de Indexação

SITEMAP .html

  • Ajuda ao usuário
  • Estrutura do Site
  • Lista de links

Rich Snippets

  • Estilo de Formatação
  • Facilita a exibição do conteúdo
  • É interessante ao usuário
  • Não foca em perfomance SEO (mas sim, UX)

Página 404

  • Importante para o usuário
  • Direcionar usuário
  • Não indexar a página
  • Dar opções ao usuário

Conteúdo

  • Principal para o SEO
  • Foco no usuário
  • Não duplicar
  • Tornar acesso fácil através do mecanismo

Breadcrumbs - Trilha de Navegação

  • Importante para o SEO
  • Mantém relação entre páginas
  • Interessante manter links ativos
  • Melhora navegação

Perfomance

  • Manter scripts no fim da página
  • Minificar js/html
  • Minificar css
  • Comprimir

Links Úteis

  • Google Adwords
  • Google Webmaster
  • Google PageSpeed Insights
  • Google Advanced SEO
